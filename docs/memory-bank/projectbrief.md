## What is onitask?

onitask is an **AI-Native Control Plane** for hybrid teams where humans and autonomous AI agents coordinate as equal workers in a unified, adaptive, and overload-protected space.

### One-Liner

> **onitask is the first AI-Native Control Plane for hybrid teams (people + AI agents) — a single dashboard where both work as equal workers.**

---

## Core Problem

Most project management tools (Jira, Linear, Asana) were designed for humans only. Today, teams have AI agents (Cursor, Claude Code, custom scripts) that work 24/7 and produce tasks faster than humans can process. This leads to:

- **Chaos**: No visibility into what agents are doing
- **Overload**: Humans get buried by agent-generated work
- **Lost Context**: Tasks are discussed in Telegram, never make it to the tracker
- **Silent Failures**: Agents get stuck without escalation → wasted hours

---

## Core Solution

onitask provides:

1. **Worker Agnosticism**: Humans and AI agents are equal workers in the system
2. **Zero-Friction Input**: Create tasks from Telegram (voice/text) with AI parsing
3. **Cognitive Shield**: Protect humans from agent overload (Cognitive Budget)
4. **Agent Transparency**: Every agent action is logged and visible
5. **Smart Escalation**: Agents ask for human help when stuck
6. **Predictive Analytics**: See risks before they become problems
