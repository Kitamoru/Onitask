# Architecture & Decisions

## Архитектурные решения

### Routing
- Используется **App Router** (Next.js 13+), папка `src/app/`
- Файлы: `page.tsx`, `layout.tsx`, `loading.tsx`, `error.tsx`

### Стилизация
- **Tailwind CSS v4** — новый синтаксис конфигурации через CSS-переменные
- PostCSS плагин: `@tailwindcss/postcss`
